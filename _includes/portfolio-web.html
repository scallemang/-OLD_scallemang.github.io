Synbiota.com

ChrisLounds.ca

A tribute to a friend who passed away and a gift to his friends, family and partner, I repurposed Chris’ Blogspot blog using Turn.js. A labour of love. (All sincerity aside, the code could probably use some tidying up; I did this about a year after first getting started with HTML.)

SCAllemang.ca

This very site—here! This one! You’re looking at it!—was an excuse to a) learn Jekyll, b) roll a Swiss Design-inspired personal site (I was reading a book on Helvetica when I started sketching this out), and c) get to know Bootstrap a little better. A great learning experience: still plenty to improve, but I leveled up my programming concepts in thinking about how to loop through pages in the menu and assign active classes where appropriate.

Getting categories and tags to work on GitHub pages (which restricts Jekyll plugin use) was educational: this blogpost on MindDust was vital in setting this up.

Wild Bore album site (WIP)

A site for an album by Wild Bore (a project of some friends and me). The album still needs some edits and remixing. Anyway: I wanted a site where each song has its own window, in a sense. And I wanted a nice scroll effect between songs when you click the menu.

I made one version ‘branded’ to the album’s artwork, another compiling quirky images that I thought created a mood of weirdness and discomfort (some nasty CSS adds to this).

GitHub repo: Jekyll + tags and categories for GitHub pages

After struggling to set my own Jekyll page up on GH pages with working tags and categories, I took an out-of-the-box Jekyll installation and tried to add as little code as possible to enable category and tag pages on your GitHub Jekyll blog. Fingers crossed, the README will help the user get the rest of the way.
